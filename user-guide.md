# 用户手册

## 加入组织
1. 管理员会发送邀请邮件到您的报名邮箱。
2. 点击邮件中的链接，接受邀请。如果报名邮箱没有注册过 GitHub 账户，需要先注册一个账户，然后再点击接受邀请。
3. 登录 GitHub 账户，进入组织页面，确认加入。  
4. 在个人profile里，确认gh copilot权限已经开通。

## 安装和配置
1. 在 Visual Studio Code 或您的IDE 中安装 GitHub Copilot 插件。目前支持VS Code，Jetbrains，XCode，Visual Studio等IDE。
2. 登录 GitHub 账户并授权插件访问。
3. 登录时，可能会遇到github.com的网络连通性问题，这时候如果能翻墙的话，登录过程会顺利很多。

## 使用指南
1. GH Copilot功能分成代码补齐和Chat两大类。
2. 代码补齐会自动提供代码建议，按 `Tab` 键接受建议。
3. Chat会出现在IDE边栏，也可以通过ctl+i 打开inLine Chat使用。
4. 具体使用方式可以参考[GitHub Copilot 官方文档](https://docs.github.com/zh/copilot/using-github-copilot/getting-code-suggestions-in-your-ide-with-github-copilot)。
5. 下周会提供使用技巧和最佳实践的培训。

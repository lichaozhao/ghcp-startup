# 自助式开通GH Copilot
目标读者：GH Copilot管理员
前提：
1. 有GH账号（个人账号即可）
2. 需要用于GH Copilot计费的Azure订阅的管理员配合

## 创建GH Enterprise

1. 登录GH账号，访问[GH Enterprise页面](https://github.com/settings/enterprises)，点击右上角“New Enterprise”，创建新的Enterprise。 
2. 选择“Enterprise with personal accounts”类型的Enterprise。
3. 输入必要的Enterprise信息，注意“Enterprise Name”是企业的唯一标识符，建议使用与组织相关的名称。
4. 设置billing信息
